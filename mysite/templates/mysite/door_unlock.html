Smart Door Unlocking System

In this project, I will show how to make a smartphone-controlled, internet-connected door powered by a Raspberry Pi that can be added onto your existing door lock without any modifications to the door. The door lock can be controlled by multiple smartphones, and even notify you whenever someone locks/unlocks the door. All of this is done using a simple, drag-and-drop app builder which makes building your own Internet of Things (IoT) app easier than ever. For an extra impressive door lock, we'll also show you how to add a simple LED light to indicate if the door is locked, and a push-button to operate the lock manually.

This project will include the following modules


  • User Login
  • Camera and finger print
  • Detection
  • Lock/Unlock
  • Live Feed
  • E-Mail

Hardware Requirements

  • Raspberry pi 3 model B
  • Raspberry pi 3 Camera module
  • Servo Motor
  • Finger Print Scanner

Software Requirements

  • Open CV
  • Putty
  • Raspbian Os
